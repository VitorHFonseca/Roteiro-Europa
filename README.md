# Roteiro Europa — Hospedagem Add Fix

## Correção

O botão **Adicionar hospedagem** não funcionava quando o navegador carregava HTML antigo com JS novo.

Agora o app aceita os dois formatos:

- formulário antigo com campo **Custo**
- formulário novo com **Valor / Moeda / Câmbio / Cobrança**

Também foi adicionado tratamento de erro visível no toast para não falhar em silêncio.
